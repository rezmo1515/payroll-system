# سامانه حقوق و دستمزد پرسنل (نسخه نمونه بدون وابستگی خارجی)

این مخزن شامل یک نمونه حداقلی از سامانه حقوق و دستمزد فارسی است که ساختار و نماهای آن بر اساس پروژه [PayrollSystem](https://github.com/helloKeyur/PayrollSystem) بازطراحی شده و تمامی ماژول‌های موجود در تصویر مرجع را پوشش می‌دهد. به دلیل محدودیت‌های محیط (عدم دسترسی به پکیج‌های اینترنتی) پیاده‌سازی فعلی بدون وابستگی به **Laravel** اصلی انجام شده، اما ساختار ماژول‌ها، کنترلرها، مسیریابی و لایه نمایش به‌صورت مشابه و قابل انتقال به پروژه لاراول واقعی طراحی شده‌اند.

## پیش‌نیازها

- PHP 8.2 یا بالاتر به‌همراه افزونه `intl`

## اجرای پروژه

```bash
php -S localhost:8000 -t public
```

سپس در مرورگر به آدرس `http://localhost:8000` مراجعه کنید. داشبورد سامانه، لیست ماژول‌ها و فیش حقوقی نمونه در دسترس خواهد بود.

## ساختار شاخه‌ها

```
app/
  Http/Controllers/     کنترلرهای داشبورد و فیش حقوقی
  Models/               مدل‌های دامنه (پرسنل)
  Modules/              مدیریت ماژول‌های سامانه و توصیف امکانات
  Services/             منطق محاسبه حقوق و DTO ها
  Support/              توابع و فرمت‌کننده فارسی
  View/                 لایه نمایش ساده
bootstrap/              راه‌اندازی و بارگذاری مسیرها
config/                 تنظیمات سامانه و جدول ماژول‌ها
public/                 نقطه ورود تحت وب و فایل‌های استایل
resources/views/        قالب‌های RTL و فارسی مطابق مرجع
routes/web.php          تعریف مسیرهای اصلی
```

## ماژول‌های پیاده‌سازی شده

تمامی ماژول‌های موجود در تصویر مرجع به‌صورت داده‌محور در `config/modules.php` تعریف شده‌اند. نماهای `resources/views/dashboard.php` و `resources/views/modules/index.php` جدول کامل امکانات را مشابه نمای موجود در مخزن مرجع ارائه می‌کنند.

## توسعه و استقرار روی Laravel واقعی

- ساختار پوشه‌ها و کنترلرها با الگوی لاراول سازگار است و می‌توان به‌سادگی آن را داخل پروژه Laravel 10/11 کپی و از سرویس‌ها و کانفیگ‌ها استفاده کرد.
- کافی است کلاس‌ها را به فضای نام پیش‌فرض لاراول منتقل کرده و رندر نماها را با Blade جایگزین کنید.
- فایل‌های تنظیمات می‌توانند بدون تغییر به دایرکتوری `config/` در لاراول اضافه شوند.
